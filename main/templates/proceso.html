Nuestro Proceso de Servicio

1. Solicitud de Servicio

El cliente nos contacta para solicitar un servicio.

2. Coordinación

Nos ponemos en contacto y coordinamos el retiro del dispositivo.

3. Diagnóstico

Realizamos un diagnóstico detallado del dispositivo. El costo de retiro + diagnóstico es de $10,000, que se incluirá en el precio total si se procede con la reparación.

4. Propuesta de Solución

Presentamos al cliente una propuesta de solución con el costo estimado.

5. Decisión del Cliente

El cliente decide si proceder con la reparación o no.

6. Reparación o Devolución

Si el cliente acepta, procedemos con la reparación. Si no, devolvemos el dispositivo.

7. Pruebas Finales

Realizamos pruebas exhaustivas para asegurar el correcto funcionamiento.

8. Entrega

Entregamos el dispositivo reparado al cliente.
